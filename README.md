# Impulse

Pulseaudio visualizer module for Ruby.

## Dependencies

* Ruby >= 1.9.3
* pulseaudio (libpulse-dev)
* fftw3 (libfftw3-dev)

## API

Impulse::Engine#start
Impulse::Engine#stop
Impulse::Engine#set_source
Impulse::Engine#get_sample
